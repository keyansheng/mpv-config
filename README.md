# Install (PowerShell/Bash)

```bash
# Linux/macOS
cd ~/.config

# Windows
cd ~/AppData/Roaming

# all operating systems
mv mpv mpv-backup
git clone https://github.com/keyansheng/mpv-settings mpv
```

# Update (PowerShell/Bash)

```bash
# Linux/macOS
cd ~/.config/mpv

# Windows
cd ~/AppData/Roaming/mpv

# all operating systems
git pull
```
