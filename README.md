# Install

```bash
# Linux/macOS
git clone https://github.com/keyansheng/mpv-config $HOME/.config/mpv

# Windows
git clone https://github.com/keyansheng/mpv-config $HOME/AppData/Roaming/mpv
```

# Update

```bash
# Linux/macOS
cd $HOME/.config/mpv
git pull

# Windows
cd $HOME/AppData/Roaming/mpv
git pull
```
